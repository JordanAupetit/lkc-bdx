
Certaines options ne sont pas des booléens (true / false) ou des tristate (true / false / module) et ne doivent pas être affichées
dans notre application puisqu'elles ne sont pas modifiable pour l'utilisateur.

Le script python permet :

 - De savoir si une option est visible ou pas (Elle est visible si sa dépendance est valide et si elle est de type booléen ou transiant)
 - De connaitre l'état d'une option (true / false / module)
 - De connaitre les dépendances d'une option*


* Attention, seule les options mentionnés dans la dépendance sont affichées, mais pas les opérateurs booléen les séparant.
Cela n'est pas suffisant pour connaitre la ou les options empéchant la validation de la dépendance.
